# Swoleaf 🍃
https://swoleaf01.netlify.app/

A personal strength app for tracking those sweet gains 💪

### Lifts
- Squat
- Deadlift
- Bench press
- Pull ups

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
